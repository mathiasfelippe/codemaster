# 👨‍💻 CODEMASTER

## 💻 Sobre o Projeto

O **CODEMASTER** é uma Landing Page moderna e responsiva desenvolvida com foco de interações com JavaScript. O projeto simula a página de um Desenvolvdor de Sistemas, apresentando as ferramentas com que ele trabalha, projetos e contato.

Este projeto foi desenvolvido como parte do currículo do curso **Técnico em Desenvolvimento de Sistemas** da Escola **SENAI A. Jacob Lafer**. 

## Estado inicial do projeto
O HTML e o CSS do projeto já estão finalizados e o layout do site é exibido corretamente no navegador. Ao abrir o arquivo index.html utilizando o Live Server do VS Code, a página carrega normalmente e a estrutura visual aparece organizada.
Entretanto, algumas funcionalidades ainda não estão funcionando porque dependem de código em JavaScript, que ainda não foi implementado.

O menu mobile, links, formulário com API, animação do carrossel, modo escuro/claro e certas interações da página ainda não possuem comportamento dinâmico.
